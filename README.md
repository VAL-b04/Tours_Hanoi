# Tours_Hanoi

Un **jeu des Tours de Hanoï** développé en **Java**, combinant une interface graphique **StdDraw** et une gestion des disques basée sur des **piles (Stack)**

---

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/VAL-b04/Tours_Hanoi
cd Tours_Hanoi

# Compiler les fichiers
javac StdDraw.java ToursHanoi.java

# Lancer le jeu
java ToursHanoi
```
