# Tours_Hanoi

**Jeu des Tours de Hanoï** interactif en **Java**, utilisant **StdDraw** pour l’interface graphique.

---

## Installation

```bash
# Cloner le dépôt
git clone [https://github.com/your-username/Tours_Hanoi.git](Tours_Hanoi)
cd Tours_Hanoi

# Compiler les fichiers
javac StdDraw.java ToursHanoi.java

# Lancer le jeu
java ToursHanoi
```
